# bike_store_sales_analysis
🚲 Bike Store Sales Analysis
Data Science | Exploratory Data Analysis (EDA) | Python
This project focuses on analyzing sales data from a bike store to uncover key business insights. By using Python, I explored customer behavior, product performance, and revenue trends to help drive data-backed decision-making.

📌 Project Overview
The goal of this analysis was to process raw sales data and transform it into actionable insights. The project covers:

Data Cleaning: Handling missing values and ensuring data consistency.

Exploratory Data Analysis (EDA): Visualizing distributions and correlations.

Sales Trends: Analyzing monthly and yearly revenue growth.

Product Insights: Identifying top-performing categories and specific bike models.

🛠️ Tech Stack & Tools
Language: Python

Libraries: * Pandas (Data Manipulation)

NumPy (Numerical Analysis)

Matplotlib & Seaborn (Data Visualization)

Environment: Jupyter Notebook

📊 Key Insights
Revenue Drivers: Identified which product categories contribute most to the total revenue.

Seasonal Trends: Found peak sales periods during the year, helping in inventory planning.

Customer Demographics: Analyzed which age groups or regions are the primary buyers.

📂 Project Structure
bikes.ipynb: The main Jupyter Notebook containing all the Python code, cleaning steps, and visualizations.
